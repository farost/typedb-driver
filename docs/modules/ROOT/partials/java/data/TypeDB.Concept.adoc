=== TypeDB.Concept

`class`

*Package*: `com.typedb.driver`

Factory class to instantiate new ``Value`` concepts from raw values. These can then be used in the ``GivenRows`` of a query.

Constructs a ``GivenRows`` instance from a list of rows given as plain Java objects. Each map entry value may be a ``Concept`` (used directly) or a raw host-language value which is converted via <<#_tryConvertToValue_java_lang_Object,``tryConvertToValue(java.lang.Object)``>>.
